Installation Instructions

In order to access the SJC vanilla Minecraft server, you merely need to install Minecraft normally on the newest version. Go to the Minecraft website (https://minecraft.net/en-us/), purchase and/or download the game, and run the launcher. By default, it should run on the newest version of the game, but if you have changed anything or know what you are doing, you can confirm the version is the same as displayed above. The server can be accessed if you have other mods installed, as long as they are client-side only, but cheating mods (such as XRay) are not allowed. However, no mods of any sorts are necessary, and you should be able to connect without changing anything. Start the game, and click the "Multiplayer" button. Then click to add a server, and enter the IP address given above along with whatever name you would find memorable. After that, simply select the server and click to join!
